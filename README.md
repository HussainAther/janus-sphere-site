# Janus Sphere Site

Static launch site for Janus Sphere Innovations and RBYRCT, ray-by-ray computed tomography.

## Netlify

Use these settings:

- Build command: leave blank
- Publish directory: `.`

The included `netlify.toml` sets the publish directory and basic security headers.

## Domain

Use this repo for:

- `janus-sphere.com`
- `www.janus-sphere.com`

Keep the RBYRCT WebGL demo separate, ideally at:

- `demo.janus-sphere.com`

Do not remove Google Workspace DNS records for email.
